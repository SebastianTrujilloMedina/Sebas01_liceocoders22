Etiquetas Html

Cualquier cosa entre las etiquetas de apertura y cierre se interpretará como código HTML. Sin estas etiquetas, es posible que los navegadores interpreten incorrectamente su código HTML. Ejemplo:

Utilizando la etiqueta pre se preformatea el texto

br

El elemento HTML line break produce un salto de línea en el texto (retorno de carro). Es útil para escribir un poema o una dirección, donde la división de las líneas es significativa.

p

La etiqueta en HTML define un párrafo. Estas tienen etiqueta de apertura y cierre. Por lo tanto, todo lo mencionado se trata como un párrafo.

"b" o "strong"

Una diferencia clave es que las etiquetas deben utilizarse cuando el texto tiene una fuerte importancia o un sentido de urgencia o seriedad.

"u" o "ins"

El elemento HTML ins contiene texto que ha sido insertado en el documento. Es útil cuando los autores necesitan mostrar partes del contenido

"i" o "em"

El elemento HTML "em" es el apropiado para marcar con énfasis en las partes importantes de un texto.

h1, h2, h3, h4, h5 y h6

Las etiquetas H1, H2, H3, H4, H5 y H6 sumamente importantes al momento de querer estructurar el contenido de una página y buscar posicionamiento en buscadores.

Small

El elemento HTML "small" hace el tamaño del texto una talla más pequeña (por ejemplo, de largo a mediano, o de pequeño a extra pequeño) que el tamaño mínimo de fuente del navegador.

big

Muestra el texto marcado con un tamaño de fuente más grande. El tamaño de la fuente puede definirse en una escala del 1 al 7, grande aumenta este valor en un punto.

Sub Sup

El elemento HTML define un fragmento de texto que se debe mostrar, por razones tipográficas, más alto, y generalmente más pequeño.

"Strike" o "del"

Muestra el texto tachado con una linea horizontal.

pre

representa texto preformateado. El texto en este elemento típicamente se muestra en una fuente fija, no proporcional, exactamente como es mostrado en el archivo.

abbr

representa una abreviación o acrónimo; el atributo opcional title puede ampliar o describir la abreviatura.

mark

representa un texto marcado o resaltado como referencia o anotación, debido a su relevancia o importancia en un contexto particular.

a

crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL.

img

El elemento de imagen HTML representa una imagen en el documento.

"ul" o "li"

ul de "lista desordenada" - lista no ordenada . crea una lista no ordenada.

blockquote

cita en bloque . Crea citas en bloque, marca las citas a otros autores o documentos.
